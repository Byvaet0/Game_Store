# GameStore

**GameStore** — это веб-приложение для управления игровым магазином, которое позволяет пользователям создавать, редактировать, удалять и просматривать игры с различными жанрами и ценами. Проект состоит из двух частей: фронтенда и бэкенда.

## Описание проекта

### Backend (API)
Backend реализован с использованием **ASP.NET Core**. Он обеспечивает взаимодействие с базой данных, в которой хранятся данные об играх и жанрах. Все операции с играми, такие как создание, редактирование и удаление, выполняются через API, используя методы HTTP.

### Frontend
Frontend был реализован с использованием **React**. Он позволяет пользователю взаимодействовать с играми, используя удобный интерфейс. React-приложение получает данные о играх и жанрах с backend API, и обновляет интерфейс динамически, обеспечивая хороший пользовательский опыт.

### Основные возможности

- **Добавление новых игр**: Пользователи могут создавать новые игры, заполняя поля с названием, жанром, ценой и датой релиза.
- **Редактирование существующих игр**: Возможность изменять информацию о уже существующих играх.
- **Удаление игр**: Пользователи могут удалять игры из системы.
- **Выбор жанра**: При создании или редактировании игры пользователи могут выбирать жанр из списка доступных жанров.
- **Просмотр списка игр**: На главной странице отображается список всех доступных игр с информацией о названии, жанре, цене и дате релиза.

### Стек технологий

- **Backend**:
  - ASP.NET Core
  - Entity Framework Core (для работы с базой данных)
  - RESTful API для взаимодействия с фронтендом

- **Frontend**:
  - React
  - React Bootstrap для стилизации
